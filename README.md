Descripción breve:

M.O.C.O. es un gestor privado de archivos personales pensado para usuarios que quieren tener el control total sobre sus datos.
Permite sincronizar, organizar y respaldar información de forma local o en red, sin depender de servicios externos ni nubes comerciales.

Lema:

“Se pega a tus archivos… y no los suelta.”

Concepto:

Autohospedado (ideal para NAS o servidor casero).

Copias automáticas locales y offline.

Interfaz web minimalista.

Control completo del usuario (sin terceros ni telemetría).

Humor serio: suena a producto estatal o laboratorio secreto, pero es tu nube personal.

Identidad visual sugerida:

Logo: una gota o esfera verde (símbolo del “moco”) con las siglas M.O.C.O. en tipografía institucional (tipo DIN o Eurostile).

Colores: verde oliva + gris oscuro, aire de “agencia oficial de tecnología experimental”.

Eslogan secundario:

“Ministerio Oficial de Copias y Organización.”
